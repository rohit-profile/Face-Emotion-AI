📊 Emotion Analytics

The Emotion Analytics module provides a clear and interactive visualization of detected emotions over time. It helps users understand patterns, monitor engagement, and analyze emotional responses for individuals or groups in real-time.

🔹 Key Capabilities

  • ✔ Real-time emotion tracking with dynamic updates.
  • ✔ Interactive bar charts displaying counts of each emotion.
  • ✔ Historical analysis to identify trends and patterns.
  • ✔ Supports multiple users and multi-face tracking in one frame.
  • ✔ Easy-to-read, color-coded dashboard visualization.

📈 How It Works

  1. The system collects emotion data from uploaded images or live video feeds.
  2. Each detected face is classified by the CNN-based emotion model.
  3. Emotion counts are stored and updated dynamically for analytics.
  4. Charts display the distribution of emotions in real-time.
  5. Users can monitor trends over time to gain actionable insights.

ℹ️ About The Project

The Facial Emotion Recognition System is an AI-powered web application that detects and classifies human emotions in real-time using a Convolutional Neural Network (CNN). It leverages deep learning and computer vision to analyze facial expressions from images or live video feeds.

🚀 Technology Stack

  • ✅ Backend: Flask (Python Web Framework)
  • ✅ Face Detection: OpenCV with Haarcascade for accurate real-time detection
  • ✅ Emotion Classification: CNN trained on thousands of labeled facial images
  • ✅ Frontend: HTML, CSS, JavaScript, Chart.js for analytics visualization
  • ✅ Deployment: Cross-platform web interface accessible via desktop or laptop browsers

😊 Supported Emotions

Angry Disgust Fear Happy Neutral Sad Surprise

📊 How It Works

  1. User uploads an image or uses live camera feed.
  2. The system detects faces in the frame using OpenCV.
  3. Each detected face is processed by the CNN model to predict emotions.
  4. Detected emotions are displayed on the dashboard along with bounding boxes.
  5. Analytics and charts provide trends and patterns over time for multiple users.
